# DerekAstro.github.io starter site

This folder contains a simple static GitHub Pages site for `https://derekastro.github.io/`.

## Files

- `index.html` — the homepage text and main research cards.
- `styles.css` — visual styling shared by the homepage and detail pages.
- `research/asteroseismology.html` — detail page for asteroseismology.
- `research/polarimetry.html` — detail page for polarimetric asteroseismology.
- `research/space-photometry.html` — detail page for WIRE/Kepler/TESS and space photometry.
- `research/stellar-activity.html` — detail page for rotation, activity, and magnetism.
- `research/side-quests.html` — flexible page for other projects.

## Uploading to GitHub using the web interface

1. Unzip this folder.
2. Go to `https://github.com/DerekAstro/DerekAstro.github.io`.
3. Click **Add file → Upload files**.
4. Drag the contents of this folder into the upload area, including the `research` folder.
5. Commit directly to the `main` branch.

If GitHub warns you that `index.html`, `styles.css`, or `README.md` already exist, choose the option that replaces/updates them.

## Editing text

- Edit homepage text in `index.html`.
- Edit the longer research descriptions in the HTML files inside the `research/` folder.
- Edit fonts, colors, spacing, and card layout in `styles.css`.
